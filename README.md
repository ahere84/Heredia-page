# Andy Heredia — Portfolio

Minimal, responsive, single-page portfolio built with plain HTML/CSS/vanilla JavaScript.

## What’s Included

- Sticky navigation with smooth scrolling + active section highlight
- Light/Dark mode toggle (saved in `localStorage`)
- Project cards linking out to posts
- Gallery section with captions
- Fully static — no build tools, frameworks, or external libraries

## Local Preview

Open `index.html` directly, or run a local server:

```bash
python3 -m http.server 5500
```

Then visit `http://localhost:5500`.

## Deploy (GitHub Pages)

1. Push this repo to GitHub.
2. Go to **Settings → Pages**
3. Set **Source** to **Deploy from a branch**
4. Select **Branch:** `main` and **Folder:** `/(root)`
5. Save and wait for the deployment URL to appear.

## Notes

- Images live in `assets/`.
- Update content in `index.html`.
